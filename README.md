# CO2 Emissions Predictor

**Mission:** Predict vehicle CO2 emissions to help reduce environmental impact from transportation.

**Dataset:** CO2 Emissions Canada (Kaggle) — 7,385 real vehicle records from the Government of Canada with features like engine size, fuel type, cylinders, and transmission. Source: https://www.kaggle.com/datasets/debajyotipodder/co2-emission-by-vehicles
